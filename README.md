# Spotify-backend



Built a RESTful backend API for a music streaming platform using Node.js, Express, and MongoDB. Implemented user authentication with JWT and Bcrypt, created APIs for music upload, album management, and track retrieval. Integrated ImageKit for cloud file storage and implemented role-based access control with custom middleware.

Tech Stack: Node.js | Express.js | MongoDB | JWT, Bcrypt | ImageKit | REST APIs.
